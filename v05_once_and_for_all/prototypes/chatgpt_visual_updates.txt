# 🚦 1. Make direction (arrows) clear in `digraph`

MATLAB already draws arrows for `digraph`, but they’re often too subtle. You can make them much clearer by tweaking edge properties.

### ✅ Update your plotting section

Right after:

```matlab
obj.hPlot = plot(obj.G);
```

Add:

```matlab
% Improve directed edge visibility
obj.hPlot.ArrowSize = 12;      % bigger arrows
obj.hPlot.LineWidth = 1.5;     % thicker edges
obj.hPlot.EdgeAlpha = 0.7;     % slightly transparent
```

# 🚌 2. Smooth bus animation (instead of jumping)

Right now buses "teleport" between nodes:

```matlab
positions = [buses.currentNode];
```

To animate smoothly, you need:

* current node
* next node
* progress (0 → 1 along edge)

---

## ✅ Step 1: Add bus interpolation data

Each bus should have:

```matlab
bus.currentNode
bus.nextNode
bus.progress   % value between 0 and 1
```

---

## ✅ Step 2: Get node coordinates

```matlab
X = obj.hPlot.XData;
Y = obj.hPlot.YData;
```

---

## ✅ Step 3: Compute smooth positions

Replace your bus highlighting logic with:

```matlab
% ---------------------------
% 3. DRAW MOVING BUSES
% ---------------------------
hold on;

X = obj.hPlot.XData;
Y = obj.hPlot.YData;

for i = 1:length(buses)
    b = buses(i);

    x1 = X(b.currentNode);
    y1 = Y(b.currentNode);

    x2 = X(b.nextNode);
    y2 = Y(b.nextNode);

    % Linear interpolation
    xb = (1 - b.progress)*x1 + b.progress*x2;
    yb = (1 - b.progress)*y1 + b.progress*y2;

    plot(xb, yb, 'ro', 'MarkerSize', 8, 'MarkerFaceColor', 'r');
end

hold off;
```

---

## ✨ Result

Instead of:

* 🔴 jumping node → node

You get:

* 🟢 smooth movement along edges
